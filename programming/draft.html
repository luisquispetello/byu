Cliente/Servidor
Es un modelo de comunicación entre dos programas informáticos en el que uno (el cliente) solicita servicios o recursos a otro (el servidor), que los provee. Por ejemplo, un navegador web es un cliente que solicita páginas web a un servidor web.

certificados de seguridad
Los certificados de seguridad son documentos digitales que autentican la identidad de un sitio web y cifran la comunicación entre el navegador del usuario y el servidor, garantizando la privacidad y la integridad de los datos.

DHTML
Es la abreviatura de HTML dinámico, que se refiere al código que permite crear páginas web interactivas y que se modifican conforme el usuario interactúa con ellas. DHTML combina el uso de HTML, CSS, JavaScript y el modelo de objetos del documento (DOM).

ECMAScript
ECMAScript es el estándar que define la sintaxis y las características de JavaScript, utilizado por los navegadores web y otros entornos de ejecución para JavaScript.

FTP
FTP (Protocolo de Transferencia de Archivos) es un protocolo utilizado para transferir archivos entre computadoras a través de una red, comúnmente utilizado en el desarrollo web para cargar archivos a un servidor.

filtro SmartScreen
El filtro SmartScreen es una característica de seguridad en algunos navegadores que ayuda a proteger a los usuarios al bloquear sitios web maliciosos o fraudulentos.

Gecko
Es un motor de renderizado de HTML de código abierto, desarrollado por Mozilla, que se encarga de transformar el código de las páginas web en el contenido visual que se muestra en el navegador. Gecko es utilizado por navegadores como Firefox, SeaMonkey o Thunderbird.

Hojas de estilo CSS
CSS son las siglas en inglés de cascading style sheets, que significa hojas de estilo en cascada. El CSS es una función que se agrega al HTML para definir el aspecto visual y el diseño de las páginas web. El CSS permite separar el contenido del formato y aplicar estilos a los elementos HTML mediante selectores.

HTTP
HTTP (Protocolo de Transferencia de Hipertexto) es el protocolo utilizado para la transferencia de datos en la web, permitiendo la comunicación entre navegadores y servidores web.

Java
Es un lenguaje de programación orientado a objetos, diseñado para crear aplicaciones que se puedan ejecutar en cualquier plataforma o sistema operativo. Java se compila en un código intermedio llamado bytecode, que puede ser interpretado por una máquina virtual Java (JVM).

Javascript
Es un lenguaje de programación interpretado, basado en el estándar ECMAScript, que se utiliza principalmente para crear páginas web dinámicas e interactivas. Javascript se ejecuta en el lado del cliente, es decir, en el navegador del usuario, y puede interactuar con el HTML, el CSS y el DOM.

Presto
Es un motor de renderizado de HTML desarrollado por Opera Software, que fue utilizado por el navegador Opera hasta la versión 12. Presto fue reemplazado por Webkit y luego por Blink, un motor derivado de Webkit.

plug-in
Un plug-in es un componente de software que se integra en un navegador web para agregar funciones adicionales, como la reproducción de contenido multimedia o la visualización de PDF.

phishing
El phishing es una técnica utilizada por ciberdelincuentes para engañar a los usuarios y obtener información confidencial, como contraseñas o información financiera, haciéndose pasar por entidades de confianza.

spoofing
El spoofing es una técnica utilizada por atacantes para falsificar la identidad o la dirección de origen de una comunicación en línea con el fin de engañar a los usuarios o eludir sistemas de seguridad.

URL/URI
URL es la abreviatura de localizador uniforme de recursos, que es una cadena que indica la ubicación o dirección de un recurso en Internet. Por ejemplo, https://www.bing.com es una URL. URI es la abreviatura de identificador uniforme de recursos, que es una cadena que identifica a un recurso en Internet mediante un nombre, una ubicación o ambos. Por ejemplo, urn:isbn:0405999832 es un URI. Una URL es un tipo de URI.

Virtual Store
Una Virtual Store (Tienda Virtual) es una plataforma en línea que permite a los usuarios comprar productos o servicios a través de internet, sin necesidad de visitar una tienda física.

Webkit
Es otro motor de renderizado de HTML de código abierto, desarrollado por Apple, que también se ocupa de convertir el código de las páginas web en el contenido visual que se muestra en el navegador. Webkit es utilizado por navegadores como Safari, Chrome o Opera.
Volver arriba
widgets
Los widgets son pequeñas aplicaciones o componentes interactivos que se pueden incorporar en páginas web para proporcionar funcionalidades específicas, como relojes o calendarios.

XML
Es la abreviatura de extensible markup language, que es un lenguaje de marcado que define un conjunto de reglas para la codificación de documentos en un formato que es legible tanto por humanos como por máquinas. XML se utiliza para almacenar y transmitir datos estructurados entre diferentes sistemas.

XHTML
XHTML (Extensible Hypertext Markup Language) es una versión más estricta y extensible de HTML que sigue las reglas XML, utilizada en el desarrollo web para crear páginas bien formadas y compatibles.
